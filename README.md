# Toggle Dark/Light Mode

A polished dark/light mode toggle built for modern web interfaces.

This project is a clean, reusable implementation of theme switching with automatic system theme detection, saved user preferences, and a refined visual style inspired by the Catppuccin color palette.

## What You Get

- Dark and light mode toggle
- Automatic detection of the user's system theme
- Persistent theme preference with `localStorage`
- Responsive, modern UI
- Catppuccin-inspired light and dark themes
- Organized CSS architecture with design tokens
- Clean Lit component implementation

## Perfect For

Developers, designers, and makers who want a practical theme-switching example they can study, customize, or adapt into their own projects.
